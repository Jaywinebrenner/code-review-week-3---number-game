
# _BEEP! BOOP! Web Ap._

## Description

_Have fun with Beep! Beep! while you get Booped! and Beeped! when certain numbers are displayed_

## BDD

* _Input: efetkv_
* _Output: "Please enter a real number." _

* _Input: 7_
* _Output: 0,1,2,3,4,5,6,7_
